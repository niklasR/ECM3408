    \section{SQLite Database Creation}
    The following commands can be used to create a table called \texttt{documents}
    
    \begin{lstlisting}
$ sqlite3 skydrive.sqlite3sqlite > CREATE TABLE documents    ...> ( id INT PRIMARY KEY      ...> , message VARCHAR ); # sqlite does not impose length-limits on fields, so none is give..quit
    \end{lstlisting}
